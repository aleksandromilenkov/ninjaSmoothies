# NodeJS-ExpressJS-MongoDb-Mongoose
I created an Express app with MongoDb database using Mongoose to manipulate with the data in the collection. I implemented the MVC model. I'm using bcrypt to crypt the passwords in the database, cookie-parser to work better with cookies, jsonwebtoken to create tokens.  
I've implemented:  
- Auth routes with controllers  
- 7 views with ejs  
- User Model  
- Mongoose validation  
- Mongoose hooks  
- Cookies  
- JSON Web Tokens for logging in and out  
- Protected routes  
- Conditional rendering based on current user
